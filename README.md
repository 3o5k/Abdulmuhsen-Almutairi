## Featured Projects

### 1. Oxidative Stress Biomarker Analysis in Atherosclerosis
> Statistical and multivariate analysis of 5 serum 
> antioxidant biomarkers (MDA, GSH, GpX, GRD, SOD) 
> across 5 experimental groups using Python.
> Includes ANOVA, Tukey HSD, PCA, correlation network, 
> and linear regression.

### 2. Atherosclerosis Transcriptomics — GEO Dataset Analysis  
> Differential gene expression analysis on real 
> RNA-seq data from NCBI GEO (GSE57691).
> Pathway enrichment of oxidative stress genes 
> in human atherosclerotic aortic tissue.

### 3. Antioxidant Gene Network Analysis
> Protein-protein interaction network of SOD, GPX, 
> GRD, GSH biosynthesis pathway genes.
> Hub protein identification and network topology analysis.

### 4. Cardiovascular Risk Prediction — Machine Learning
> ML models (Random Forest, Logistic Regression, XGBoost) 
> for CVD risk classification on Cleveland Heart Disease dataset.
> Feature importance and SHAP value analysis.

### 5. NAC Drug Repurposing — Target Network Analysis
> Computational drug-target network analysis for 
> N-Acetylcysteine and related thiol compounds 
> in cardiovascular disease pathways.
